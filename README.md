# 🎸 OpenFret

**Learn guitar by fighting monsters.** OpenFret is a browser-based guitar platform where real note detection powers an RPG — no downloads, no cables, no monthly subscription.

🔗 **[openfret.com](https://openfret.com)** · 🎮 **[Try Guitar Quest Free](https://openfret.com/game/demo)** · 💬 **[Join the Discord](https://discord.com/invite/G3Pur3PzZm)**

---

## ⚔️ Guitar Quest

The core of OpenFret. Play your real guitar — the game listens through your browser microphone, detects every note in real time, and turns your practice into an RPG adventure.

- **Every correct note** deals damage to monsters
- **Wrong notes** cost you health
- **Defeating enemies** earns XP, gold, loot, and character upgrades

### Five Game Modes

| Mode | What You Learn |
|------|---------------|
| ⚔️ **Attack** | Fretboard note positions |
| 🔨 **Crafting** | Note patterns & finger movement |
| ✨ **Magic** | Scales, modes & musical patterns |
| 📖 **Wisdom** | Music theory — intervals, chords, circle of fifths |
| 🎤 **Gigging** | Performance challenges that test everything |

---

## 🛠️ Platform Features

- **Smart Guitar Inventory** — Add your guitars with auto-filled specs from thousands of models. Track woods, pickups, tunings, and string change history.
- **Practice Analytics** — Persistent timers, metronome, scale visualization, song logging with Last.fm integration.
- **Session Mode** — Real-time music collaboration. Fork sessions, layer tracks, and build together — like GitHub for music.
- **Guitar Tuner** — Free chromatic tuner with sharp/flat detection. Always free, no account needed.
- **Community** — Share your gear, discover players, and connect via Discord.

## 🎛️ What You Need

- Any guitar (acoustic or electric)
- A browser (Chrome, Firefox, Safari, or Edge)
- A microphone (laptop mic works for acoustic; audio interface recommended for electric)
- No downloads. No plugins. Setup takes ~2 minutes.
